7   /**
8    * Super class of all factories of {@link ObjectPersisterFactory} classes. They
9    * are responsible for creating {@link ObjectPersister} classes that will
10   * save/load data for a given class. Such factories are useful because we
11   * sometimes need to create {@link ObjectPersister} dynamically. For instance
12   * when we serialize items as JSON, we need to create a new
13   * {@link ObjectPersister} class for every class of items saved/loaded into
14   * cache. A unique {@link ObjectPersister} would not be able to strongly type
15   * the load/save method and we would have to cast the result of both operations,
16   * leading to less robust and less convenient code.
17   * @author sni
18   */
19  public abstract class ObjectPersisterFactory implements Persister {

27       * As no list of handled classes is specified, it will act has if it can
28       * handle all classes.
29       * @param application
30       *            the android context needed to access android file system or
31       *            databases to store.
32       */
33      public ObjectPersisterFactory(Application application) {
34          this(application, null);

38       * Creates an {@link ObjectPersisterFactory} given an Android application

41       *            the android context needed to access android file system or
42       *            databases to store.
43       * @param listHandledClasses
44       *            the list of classes that is handled by the factory. The
45       *            factory will try to produce an {@link ObjectPersister} for

48      public ObjectPersisterFactory(Application application,

58      /**
59       * Wether or not this bus element can persist/unpersist objects of the given
60       * class clazz.
61       * @param clazz
62       *            the class of objets we are looking forward to persist.
63       * @return true if this bus element can persist/unpersist objects of the
64       *         given class clazz. False otherwise.
65       */
66      @Override
67      public boolean canHandleClass(Class<?> clazz) {
68          if (listHandledClasses == null) {
69              return true;
70          } else {
71              return listHandledClasses.contains(clazz);
72          }
73      }
